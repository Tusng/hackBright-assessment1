3) Explain the difference between these two commands in the terminal:
   
   $ cd Users
   $ cd /Users

Your answer: 
cd Users is the relative path, which means the current working directory must be the parent of Users directory/file.
cd /Users is the absolute path.

8) What is the difference between `git add` and `git commit`?

Your answer:
'git add' is used for tracking files/projects.
'git commit' is used for commiting the changes(snapshots) of the tracked files/projects to the current repository.
------------------
9) What does the `git push` command do?

Your answer:
uploads the work, changes(snapshots) of files/projects from a local computer repository onto an online server or a cloud server(GitHub) by using a remote via an URL

------------------
10) When would you use the `git clone` command?

Your answer:
when there is no local repository in the current working directory and import files/projects from an online server or a cloud server via an URL
